# 📊 Análise de Vendas com SQL + LibreOffice

Projeto desenvolvido para análise de dados utilizando SQL
para tratamento e consultas e LibreOffice Calc para visualização.

## 🛠 Tecnologias
- SQL
- LibreOffice Calc
- CSV

## 📈 Análises realizadas
- Total de vendas por mês
- Produto mais vendido
- Ticket médio
